Blinkit Sales Analysis & Prediction

This project combines Power BI descriptive analytics with Machine Learning to analyze and predict grocery sales performance.

Business Intelligence (Power BI)Built an interactive dashboard to visualize key performance indicators - 

Sales Metrics: Total sales, top-performing items, and fat content distribution.
Visuals: Donut charts, bar charts, and summary cards.
Interactivity: Integrated slicers for dynamic data filtering.

Predictive Modeling (ML)Implemented a Random Forest Regressor to identify the primary drivers of revenue.

Performance: Achieved an $R^2$ of 0.57.
Top Insights: Item Visibility (35%) and Item Weight (31%) were identified as the most significant predictors of sales.

Tech Stack - 

Excel: Data cleaning and initial EDA.
Power BI: Data visualization and dashboarding.
Python (Scikit-Learn): Random Forest modeling and feature importance analysis.
